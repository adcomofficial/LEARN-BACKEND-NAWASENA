### **CLONE/DOWNLOAD REPO**

### INSTALASI

Apabila ingin menggunakan repo ini sebagai template pastikan di laptop kalian sudah terinstall nodejs. Lalu buka folder repo ini buka terminal kemudian jalankan script berikut

**with npm**

```shell
npm install
```

**with yarn**

```shell
yarn
```
